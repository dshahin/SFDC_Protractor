SFDC_Protractor
===============

Basic Protractor setup to login to any given SFDC org

* ```git clone https://github.com/dshahin/SFDC_Protractor.git```
* ```cd SFDC_Protractor```
* ```npm install```
* Set environment vars:    
  * SFDC_LOGIN_URL
  * SFDC_USERNAME
  * SFDC_PASSWORD
* ```npm test```
